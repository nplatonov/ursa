na.omitR Documentation

Drop bands which don't have data.

Description

The bands with band.blank images, are omitted.

Usage

## S3 method for class 'rmapRaster'
na.omit(object, ...)

Arguments

object

Object of class rmapRaster.

...

Ignorered.

Value

Object of class rmapRaster, which has no bands without any data.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

band.blank

Examples

session.grid(NULL)
session.grid(regrid(mul=1/4))
a <- rmap.new(value=1:3)
print(a)
a[2] <- NA
print(a)
a2 <- na.omit(a)
print(a2)
