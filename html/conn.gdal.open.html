gdal.openR Documentation

Open GDAL file

Description

gdal.open creates object of rmapRaster class, and prepares connections for data reading.

Usage

gdal.open(fname, verbose = FALSE)

Arguments

fname

Character. Filename; full-name or short-name.

verbose

Logical. verbose=TRUE provides some additional information on console. Default is FALSE.

Details

gdal.open doesn't read data. Data can be read later using Extract operator [.

If argument fname is ENVI .hdr Labelled Raster then either gdal.open or envi.open can be used. The former provides external implementation for data reading via GDAL in rgdal package.

Value

Returns object of class rmapRaster. Values are not in memory.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

close, envi.open, gdal.read.

Examples

session.grid(NULL)
fname1 <- system.file("pictures/cea.tif",package="rgdal")
message(fname1)
a1 <- gdal.open(fname1)
print(a1)
print(a1[])
close(a1)
fname2 <- system.file("pictures/test_envi_class.envi",package="rgdal")
message(fname2)
b1 <- gdal.open(fname2)
b2 <- envi.open(fname2)
print(b1)
print(b2)
print(c('The same grid?'=identical(rmap.grid(b1),rmap.grid(b2))
       ,'The same data?'=identical(rmap.value(b1[]),rmap.value(b2[]))))
close(b1,b2)
