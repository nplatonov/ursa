repR Documentation

Replicate bands of raster image.

Description

rep for object of class rmapRaster creates new rmapRaster objects with repitition of orininal band sequence.

Usage

## S3 method for class 'rmapRaster'
rep(x, ...)

Arguments

x

Object of class rmapRaster

...

Further arguments to be passed to or from other methods. Keywords:

times

Positive integer. Number of times to repeat each band.

If argument has no name, then times is assumpted.

Value

Object of class rmapRaster.

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

c for rmapRaster.

Examples

session.grid(NULL)
session.grid(regrid(mul=1/4))
a <- rmap.dummy(nband=3)
print(a)
b1 <- rep(a,by=2)
print(b1)
b2 <- rep(a,length=5)
print(b2)
b3 <- rep(a[3],3)
print(b3)
