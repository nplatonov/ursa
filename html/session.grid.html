session.gridR Documentation

Get and set sessional parameters for grid.

Description

session.grid without arguments returns current grid properties. session.grid with arguments specifies grid, which is used by functions of this package, e.g., for plotting, for opened and created raster images during current session.

Usage

session.grid(obj)
session.proj4()
session.cellsize()

Arguments

obj

Either missing, or NULL, or file name, or object of class rmapRaster, or object of class rmapGrid

Details

session.grid deals with option “rmapSessionGrid”: options(rmapSessionGrid=...) or getOption("rmapSessionGrid").

Usage session.grid() without arguments return value of “rmapSessionGrid” option via calling getOption("rmapSessionGrid"). If is.null(getOption("rmapSessionGrid")) then session.grid() returns default CRS.

Usage session.grid(NULL) resets “rmapSessionGrid” option via calling options(rmapSessionGrid=NULL).

The sequential calling 

session.grid(NULL)
session.grid()

returns default CRS. For checking that the option has been resetted successfully, use getOption("rmapSessionGrid") after session.grid(NULL)

Value

Object of class rmapGrid. It is a list. Default values are grid parameters of NSIDC polar stereo gridded data of Northern hemispere with nominal gridded resolution 25 km (http://nsidc.org/data/polar_stereo/ps_grids.html).

List of 9
 $ columns: int 304
 $ rows   : int 448
 $ resx   : num 25000
 $ resy   : num 25000
 $ minx   : num -3850000
 $ maxx   : num 3750000
 $ miny   : num -5350000
 $ maxy   : num 5850000
 $ proj4  : chr "+proj=stere +lat_0=90 +lat_ts=70.0 +lon_0=-45.0 +k=1
                 +x_0=0.0 +y_0=0.0 +a=6378273.000 +b=6356889.449 +units=m +no_defs"
 - attr(*, "class")= chr "rmapGrid"
NULL

session.proj4 returns item proj4.

session.cellsize returns squared root from multiplication of cell dimension: sqrt(resx*resy).

Author(s)

Nikita Platonov platonov@sevin.ru

See Also

Class rmapGrid. Use regrid to partial grid changing.

Examples

session.grid(NULL)
getOption("rmapSessionGrid") ## NULL
(g1 <- session.grid()) ## default
g1$resx <- g1$resy <- 12500
g1$columns <- as.integer(with(g1,(maxx-minx)/resx))
g1$rows <- as.integer(with(g1,(maxy-miny)/resy))
session.grid(g1)
session.grid(NULL)
a <- rmap.new(value=3)
session.grid(a)
