rmapValueR Documentation

Values of raster images.

Description

Class rmapValue is a part of class rmapRaster. It contains values of image. In the case of numeric values, the exterior class is rmapNumeric. In the case of categorical values, the exterior class is rmapCategory.

Usage

## S3 method for class 'rmapCategory'
print(x, ...)

## S3 method for class 'rmapNumeric'
print(x, ...)

rmap.value(obj, band)
rmap.value(obj, band) <- value

Arguments

x

Object of the one of classes rmapNumeric or rmapCategory

...

Further arguments passed to generic functions print and str.

obj

Object of class rmapRaster

band

Optional. Vector of band numbers (positive integer) or band names (character).

value

Numeric or integer scalar, vector, or matrix. Coerced to dimenstion of rmapValue.

Details

Try to use high-level assignment using replacement [<- operator for class rmapRaster. However, if you don't get desired result, you can downgrade the level of your code.

Value

Object of class rmapNumeric is a numerical matrix. Object of class rmapCategory is an integer matrix. Dimensions of this matrix:

dim(...)[1]

Spatial domain; the length is muliplications of lines (rows) and samples (columns)

dim(...)[2]

Band or temporal domain; the length is number of bands

It is allowed to use scalar value NA in the case when values are not in memory. In this case the class is rmapValue.

Examples

session.grid(NULL)
session.grid(regrid(mul=1/4))
a1 <- envi.create("exam1.envi",bandname=c("today","tomorrow"))
str(rmap.value(a1))
close(a1)
envi.remove("exam1")
a2 <- rmap.dummy(nband=4,min=1,max=99)
str(rmap.value(a2),digits=3)
a3 <- as.integer(a2)
str(rmap.value(a3))
str(rmap.value(a3,2))
print(rmap.value(a3))
print(a3)
rmap.value(a3,"Band 2") <- 199
rmap.value(a3)[,3] <- 299
a3[4] <- 399
print(a3)
rmap.value(a3[1:3]) <- rmap.value(a3[4])
print(a3)
rmap.value(a3[1:3]) <- -c(1:3)
print(a3)
