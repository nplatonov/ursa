as.integerR Documentation

Transform values to type integer

Description

as.integer for object of class rmapRaster truncates decimal part of image values and then converts to type integer.

Usage

## S3 method for class 'rmapRaster'
as.integer(x, ...)

Arguments

x

rmapRaster object

...

Other arguments which passed to function as.integer of package base.

Value

Object of class rmapRaster where storage.mode of values is integer.

Author(s)

Nikita Platonov platonov@sevin.ru

Examples

session.grid(NULL)
a <- pixelsize()
a <- a-min(a)+0.5
str(rmap.value(a))
print(storage.mode(a$value))
b <- as.integer(a)
str(rmap.value(b))
print(storage.mode(b$value))
