rmap.gridR Documentation

Extract and assign spatial paramaters of raster images.

Description

Raster image (rmapRaster) contains embedded spatial parameters (rmapGrid) in item $grid. These functions manipulate with item $grid.

Usage

rmap.grid(obj)
rmap.grid(obj) <- value

rmap.ncol(obj)
rmap.nrow(obj)
rmap.columns(obj)
rmap.rows(obj)
rmap.samples(obj)
rmap.lines(obj)

rmap.extent(obj)
rmap.bbox(obj)

Arguments

obj

rmapRaster object. For rmap.grid function list of rmapRaster objects is allowed

value

rmapGrid object

Details

rmap.grid<- may used to minor corrections of spatial parameters. Howevert, it seems that this function is not claimed in practice.

rmap.ncol, rmap.columns, rmap.samples are synonyms for extracting number of columns/samples.

rmap.nrow, rmap.rows, rmap.lines are synonyms for extracting number of rows/lines.

rmap.extent, rmap.bbox, are synonyms for extracting boundary box (spatial extent).

Value

rmap.grid return value of $grid item of rmapRaster object.

rmap.grid<- return rmapRaster with modified $grid item.

rmap.ncol, rmap.columns, rmap.samples return integer of length 1.

rmap.nrow, rmap.rows, rmap.lines return integer of length 1.

rmap.extent, rmap.bbox return numeric of length 4 (xmin, ymin, xmax, ymax).

Examples

session.grid(NULL)
a <- pixelsize()
print(rmap.grid(a))
rmap.grid(a)$proj4 <- gsub("\\.0+","",rmap.grid(a)$proj4)
print(rmap.grid(a))
